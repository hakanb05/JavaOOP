# Java OOP Assignments

This repository contains a collection of Java assignments focused on
Object Oriented Programming (OOP).

The goal of these assignments was to practice fundamental OOP concepts
such as inheritance, abstraction, polymorphism, and the use of interfaces
and collections.

Each assignment demonstrates different aspects of Java programming and
shows how object oriented design can be used to model real world systems.

---

# Concepts Demonstrated

The assignments in this repository include examples of:

- Object Oriented Programming (OOP)
- Inheritance
- Abstract classes
- Method overriding
- Polymorphism
- Interfaces
- Encapsulation
- Collections (such as ArrayList)
- Basic object modeling

---

# Example Assignment: Firework System

One of the assignments models a small firework product system.

The system includes different types of fireworks such as:

- **Vuurwerk (Firework)** – base class representing a firework product  
- **Knaller (Firecracker)** – a firework with properties like decibel level  
- **Vuurpijl (Rocket)** – a firework with height and color composition  

Features implemented in this assignment include:

- A product hierarchy using inheritance
- Abstract base classes for shared behavior
- Validation rules for legality of fireworks
- Use of collections to manage multiple products
- Object composition using instruction objects
